zen15
=====

## Install
1. Update [Google Analytics](http://www.google.com/analytics/) ID (or just remove `views/ga.dust`)
2. Create `config.js` file in `routes/` dir:
```
  module.exports = {
    client_id: 'your_facebook_app_id',
    client_secret: 'your_facebook_app_secret'
  };
```

## Dependencies
```
npm install
```

## Run
```
npm start
```
